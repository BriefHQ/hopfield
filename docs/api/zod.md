# Zod

Hopfield exports the as [Zod](https://github.com/colinhacks/zod) schemas from the `'hopfield/zod'` entrypoint.

## Install

Install the Zod peer dependency:

::: code-group

```bash [pnpm]
pnpm add zod
```

```bash [npm]
npm i zod
```

```bash [yarn]
yarn add zod
```

:::

## Usage

Import and use schemas:

```ts twoslash
import { test } from 'hopfield/config'
```
